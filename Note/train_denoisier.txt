● Giải thích bảng model.summary() của Denoiser

  3 cột chính

  ┌──────────────┬──────────────────────────────────────────────────────────┐
  │     Cột      │                         Ý nghĩa                          │
  ├──────────────┼──────────────────────────────────────────────────────────┤
  │ Layer (type) │ Tên layer + loại (Conv2D, MaxPooling2D, UpSampling2D...) │
  ├──────────────┼──────────────────────────────────────────────────────────┤
  │ Output Shape │ Kích thước tensor đầu ra: (None, H, W, C)                │
  ├──────────────┼──────────────────────────────────────────────────────────┤
  │ Param #      │ Số trọng số cần học (weights + bias)                     │
  └──────────────┴──────────────────────────────────────────────────────────┘

  ---
  Cột "Output Shape" — (None, H, W, C)

  (None, 28, 28, 4)
    │     │   │  └── C = 4 kênh (channels/filters)
    │     │   └───── W = 28 chiều rộng (pixels)
    │     └───────── H = 28 chiều cao (pixels)
    └─────────────── None = batch size (không cố định khi train)

  - Ảnh MNIST đầu vào: 28×28×1 (grayscale)
  - Sau Conv2D(4 filters): 28×28×4
  - Sau MaxPooling2D: 14×14×4 (giảm nửa H và W)
  - Sau UpSampling2D: 28×28×4 (phóng to lại)

  ---
  Cột "Param #" — cách tính

  Conv2D layer (ví dụ: 4 filters, kernel 3×3, input 1 channel):
  Params = filters × kernel_H × kernel_W × input_channels + filters (bias)
         = 4 × 3 × 3 × 1 + 4
         = 36 + 4 = 40

  MaxPooling2D / UpSampling2D:
  Params = 0  ← không có trọng số, chỉ là phép toán

  Conv2D layer cuối (1 filter, kernel 3×3, input 4 channels):
  Params = 1 × 3 × 3 × 4 + 1
         = 36 + 1 = 37

  ---
  Toàn bộ denoiser 665 params

  encoder:
    conv1:  4 filters, 3×3, input 1ch  →  4×3×3×1 + 4  =  40
    conv2:  8 filters, 3×3, input 4ch  →  8×3×3×4 + 8  = 296
    conv3: 16 filters, 3×3, input 8ch  → 16×3×3×8 + 16 = 1,168  ← (bước encode sâu nhất)

  decoder (mirror):
    conv4: 8 filters, 3×3, input 16ch  →  8×3×3×16 + 8 = 1,160
    conv5: 4 filters, 3×3, input  8ch  →  4×3×3×8  + 4 =   292
    conv6: 1 filter,  3×3, input  4ch  →  1×3×3×4  + 1 =    37

  Tổng = 40 + 296 + 1168 + 1160 + 292 + 37 = 2,993 — nếu bảng của bạn hiển thị 665 thì model nhỏ hơn (ít layers hơn),
  cần xem lại architecture cụ thể.
